# Steve Jobs — An Analytical Companion · 《史蒂夫·乔布斯传》深度解读

An independent, bilingual (English / 中文) analytical companion to Walter Isaacson's authorized biography **«Steve Jobs»** (© 2011 Walter Isaacson / Simon & Schuster). It maps the ideas that run through the book into ten themes and rebuilds them as original interactive visualizations: the intersection of art and technology, simplicity and design, the reality distortion field, end-to-end integration ("the whole widget"), focus and the power of no, the products-first creed, the 1976 → 1985 ouster → NeXT/Pixar wilderness → 1997 return arc, taste and craft, and legacy — plus a **Simplicity Studio** flagship, a product timeline, a clustered "principles" explorer, a six-voice analyst, a method radar, and a compounding "digital hub" recursion engine.

The throughline is deliberately two-handed — the book's own unresolved question: *were the brilliance and the brutality two things, or one?* This site is **independent commentary, paraphrase, and analysis — not the book**, and not affiliated with the author or publisher. Principles are restated in original words with attribution; the cost is never airbrushed.

## Stack

Next.js 14 (App Router) · React 18 · TypeScript · Tailwind CSS. All visualizations are hand-built Canvas / SVG with zero chart dependencies. Fully bilingual with a no-reload language toggle. Apple-graphite identity: Manrope + Newsreader typography, a space-gray palette, and the vintage six-stripe spectrum used sparingly.
